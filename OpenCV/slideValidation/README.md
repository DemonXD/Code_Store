### 基于OpenCV的滑动验证图片识别

**模板匹配识别缺口**  
具体是实现过程如下：
- 1.处理滑块的图片
    - 灰度化滑块图片
    - 处理一下滑块图中滑块的外圈
    - 使用inRange二值化滑块图
    - 使用开运算去除白色噪点

- 2.处理带缺口的图片
    - 先来个高斯滤波去噪
    - 灰度化带缺口图
    - 使用阈值二值化该图

**轮廓检测识别缺口**  
基于轮廓检测缺口的思路简单很多，加上合理的条件识别率在95%以上，实现过程如下：
- 带缺口图高斯模糊去噪
- 用(200,400)的阈值做Canny边缘检测
- 寻找轮廓
- 对已有的轮廓做约束，比如轮廓的面积范围，轮廓的周长范围
